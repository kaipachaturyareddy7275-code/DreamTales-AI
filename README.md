# 🌙 DreamTales AI

An AI-powered Personalized Bedtime Story Generator that creates magical bedtime stories for children using Google Gemini AI.

## 📖 Project Overview

DreamTales AI generates unique bedtime stories based on:

* Child's Name
* Age
* Favorite Character
* Story Theme / Moral
* Story Style
* Story Length

The generated stories are age-appropriate, creative, and include positive moral lessons.

## 🚀 Installation

### Clone Repository

git clone https://github.com/kaipachaturyareddy7275-code/DreamTales-AI.git

### Install Dependencies

pip install -r requirements.txt

### Create .env

GEMINI_API_KEY=YOUR_API_KEY

### Run Backend

cd backend

python app.py

### Open Frontend

Open:

frontend/index.html
